# Papers

The primary source(s) being formalized, plus literature notes.

- **Commit the compiled PDF** of the source paper here (`<source>.pdf`) so the
  exact statements are versioned with the formalization.
- Keep raw TeX/source in a `*-source/` subdir — it's gitignored (reference only;
  the PDF is the committed artifact).
- Cite the source's equation/section labels in each Lean docstring (a CLAUDE.md
  convention) so a reader can audit the Lean against this PDF.

`README.md` of the project carries the *what-is-formalized* table mapping each
source reference to its Lean declaration.
